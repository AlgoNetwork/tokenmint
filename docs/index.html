How it works

PREREQUISITE

You need a web3.0 enabled Ethereum wallet in order to create your tokens. We recommend using Metamask.

CREATING TOKENS

Use our token generation wizzard to complete the process.

  • We will create a smart contract on Ethereum network using address you provided.
  • We will top up the contract with your tokens.
  • You are always in control of your funds and tokens.
  • We will never ask you for your private key.
  • We charge a fee for the service we provide.
